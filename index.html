Bus Schedule Checker

What is this?

You can check the frequency and schedule of all RapidKL Buses and MRT Feeder Buses on this website. It pulls GTFS Static data daily from data.gov.my and displays the bus schedules. The accuracy of this data is based on the GTFS Static Data and the respective operator, such as RapidKL.

Easy to use

Using this schedule checker is easy, and you can also track bus locations in real time with the integration of MyRapid - Bus Kiosk on the website.

Data Accuracy

Data is pulled daily from the data.gov.my GTFS Static API at 1:00 AM, ensuring that the latest data for the day is retrieved and processed.
